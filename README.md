# 🏛️ Full-Stack Student Management & AI Training Web Portal

A secure, high-performance web portal engineered to manage academic databases and track machine learning training pipelines. Originally structured as a static frontend layout, this system is being modularized into a full-stack architecture powered by a **Python (FastAPI)** core service layer.

---

### 🚀 Technical Engineering Overhaul

* **Python Backend Migration:** Transitioned client-side routing into a structured, production-ready Python API gateway.
* **Glassmorphism Analytics:** Implemented a unified system layout that blends complex data tables with reactive, clean architectural widgets.
* **AI Pipeline Tracking:** Integrated structured schemas to log real-time training steps, epoch iterations, and validation metrics for deployed LLMs.
* **Secure API Protocol:** Configured for Pydantic data modeling to ensure zero-trust structural parsing across portal forms.

---

| Architecture Layer | Core Component & Engine |

### 🛠️ Production Local Initialization

Deploy the administrative API layer using the following pipeline:

1. **Clone the master architecture:**
```bash
   git clone [https://github.com/Mahnoor-fatima249/My-Web-Portal.git](https://github.com/Mahnoor-fatima249/My-Web-Portal.git)
   cd My-Web-Portal

```

2. **Set up virtual environment & requirements:**

```bash
   pip install -r requirements.txt

```

3. **Boot the gateway application server:**

```bash
   uvicorn main:app --reload --port 8000

```
